# Hướng Dẫn Tải và Chạy SillyTavern 🚀

SillyTavern là một công cụ tuyệt vời để tương tác với các mô hình AI như LLM (Large Language Models) và các công cụ tạo ảnh, giọng nói. Bài viết này sẽ hướng dẫn bạn cách tải, cài đặt và chạy SillyTavern một cách dễ dàng trên nhiều nền tảng khác nhau. Cùng bắt đầu nhé! ✨

## 1. Giới Thiệu Về SillyTavern

SillyTavern là một giao diện người dùng **miễn phí** và **mã nguồn mở**, giúp bạn kết nối với các mô hình AI phổ biến như OpenAI, KoboldAI, NovelAI và nhiều hơn nữa. Nó hỗ trợ đa nền tảng, bao gồm:

- Windows
- macOS
- Linux
- Android

Với SillyTavern, bạn có thể tạo ra những cuộc trò chuyện thú vị, tạo ảnh, giọng nói và nhiều tính năng khác. 🎭

## 2. Cách Tải và Cài Đặt SillyTavern

### Bước 1: Tải Xuống

Bạn có thể tải SillyTavern từ các nguồn sau:

- **GitHub (phiên bản mới nhất):**  
  Truy cập [trang GitHub của SillyTavern](https://github.com/SillyTavern/SillyTavern) và tải file ZIP hoặc clone repository.

- **SourceForge (bản mirror):**  
  [Tải tại đây](https://sourceforge.net/projects/sillytavern.mirror/) nếu GitHub bị giới hạn.

### Bước 2: Giải Nén

Sau khi tải về, giải nén file ZIP vào thư mục bạn muốn cài đặt. 📂

### Bước 3: Chạy SillyTavern

Tùy thuộc vào hệ điều hành của bạn, hãy làm theo các bước sau:

- **Trên Windows:**  
  Mở file `Start.bat` hoặc `Start.cmd` trong thư mục giải nén.

- **Trên macOS/Linux:**  
  Mở Terminal, điều hướng đến thư mục và chạy lệnh:
  ```bash
  ./start.sh
  ```

- **Trên Android:**  
  Xem hướng dẫn chi tiết trên YouTube: [Video hướng dẫn cài đặt cho Android](https://www.youtube.com/watch?v=5WQuTeNB5l4).

## 3. Cấu Hình Ban Đầu

Sau khi chạy SillyTavern, bạn cần thực hiện một số cấu hình cơ bản:

1. **Kết nối với API AI:**  
   - Chọn API từ menu (OpenAI, KoboldAI, v.v.).
   - Nhập API key nếu cần.

2. **Tùy chỉnh giao diện:**  
   SillyTavern cho phép bạn thay đổi theme, font và nhiều tùy chọn khác để phù hợp với sở thích cá nhân. 🎨

Nếu bạn muốn xem hướng dẫn trực quan, hãy tham khảo các video sau:

- SillyTavern hoàn toàn miễn phí, nhưng một số API (như OpenAI) có thể yêu cầu phí.
